# TSP & Dijkstra Delivery Planner

A Java Swing-based application to plan optimal delivery routes using **Dijkstra's algorithm** and **Traveling Salesman Problem (TSP)**. 

1] Features
- Enter up to 7 custom spots along the journey.
- Set pairwise distances manually.
- Select locations to visit and calculate optimal route.
- Visual map showing route path and total distance.
- Combines shortest path computation and TSP optimization.

2] Tech Stack
- Java, Swing for GUI
- Core Algorithms: Dijkstra, TSP

3] How to Run
1. Clone the repo:git clone https://github.com/shreya-lingayat/MargDarshak_TSP_Djikstra_Planner.git
2. 2. Open in VS Code or any Java IDE.
3. Compile and run `TSPPlanner.java`.
